## What to build

Add the controller building — the final craftable structure that anchors the win condition. It is crafted (not discovered), so the recipe becomes satisfiable once the player has visited all five planets.

**Recipe**: `3 computronium + 3 unobtainium + 1 Radar Station Core Sample + 1 Factory Core Sample + 1 Inferno Core Sample + 1 Empire Core Sample + 1 Alien Lair Core Sample`.

The controller building is placed on the home base. It accepts an astronaut stacked on it as part of the win condition (see `issues/015-win-condition.md`). No milestone unlock needed — the recipe becomes craftable naturally once ingredients are available.

## Acceptance criteria

- [ ] Controller building card and recipe exist
- [ ] Recipe requires all five core samples plus 3 computronium and 3 unobtainium
- [ ] Controller building can be placed on any board (intended for home base)
- [ ] Building can have an astronaut stacked on it
